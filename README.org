* Just a personal R package

Convenience and utility functions for personal use. Not to be published anywhere.

How to edit :

- edit =R/functions.r=
- =library(devtools)=
- =load_all()=
- test
- =install()=

To install on a new system :

- =git clone https://...=
- launch R in package directory
- =library(devtools)=
- =install()=
